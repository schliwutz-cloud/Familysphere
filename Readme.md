## Vision

FamilySphere ist eine moderne Open-Source-Plattform zur Verwaltung von Familien, Ahnenforschung und Familiengeschichte.

Sie kombiniert klassische Genealogie mit moderner Softwareentwicklung, KI-Unterstützung und einer intuitiven Benutzeroberfläche.

---

## Ziele

- Dynamischer Stammbaum
- Personenverwaltung
- Beziehungen
- Familienchronik
- Fotoarchiv
- Dokumentenarchiv
- Zeitstrahl
- Karten
- KI-Unterstützung
- GEDCOM Import/Export
- Mehrbenutzerbetrieb
